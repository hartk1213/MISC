![1](Images/1.png) 

First secure the z joint to the mgn9H carriage

![2](Images/2.png)

Then secure it to the gantry like any other Z joints, main difference is the m5 screw gets replaced with an M3 , so you need to replace the tnut in the end of the Y extrusions with an M3 tnut

![1](Images/3.png) 

Now if you are using the Y endstop version you need to solder the wires to the switch before it gets installed 

![1](Images/4.png) 

insert the switch with the lever facing up 

![1](Images/5.png) 

use 2 pieces of filament to secure the switch in place

![1](Images/7.png)

remove the 4 M3x6 installed in Step 1 and install the endstop housing using 4 m3x12 SHCS
